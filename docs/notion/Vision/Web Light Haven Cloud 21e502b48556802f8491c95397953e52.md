# Design Doc

Create an educational 2D platformer game using Phaser 3.7+ with Matter.js physics, inspired by Typing.com's simple games that using the keyboard as its input, BUT we will use voice instead.

**Design Vision:**

- Clean, modern interface with Duolingo-inspired bright colors
- Educational voice mechanics integrated into platformer gameplay
- React shell hosting the Phaser game instance
- Engaging animations and progress tracking

**First Version Features:**

1. Phaser game canvas embedded in React app shell using https://github.com/microsoft/onnxruntime/tree/main/js/web
2. Matter.js physics-based platformer character
3. Type words to perform actions (jump, move, etc.)
4. Clean UI with score tracking and progress indicators
5. Basic level with platforms and collectibles

**Design System:**

- Primary: Bright educational green (Duolingo-inspired)
- Accent: Vibrant blue and orange
- Modern gradients and smooth animations
- Rounded, friendly UI elements
