# Light Haven Home

<aside>
✅

Mobile App for parents to:

- See the progress of how they are doing in the game
- Add or remove homeschool activities
- Reward children in the game by doing non-game activities
- Assign a child in the game to learning how to cook, clean, listen to parents, and then be rewarded with in-game rewards
- Control how much time they get to play on the PC and on the Web
- Setup a schedule for homeschool
- Manage the type of subjects you want them to focus on
- **Potentially tell the game what next learning to focus on next??,**
- Allow them to set limits on game play
- Add out-of-the-game tasks that are treated like Side Quests
- Perhaps allow them to add Critical Quests themselves that involve out-of-the-game tasks that are added to the list of moving onto the story
    - focus more on this sound, or this subject again, they need more recall
- Send feedback to the developers on what should be added
</aside>

<aside>
✅

The parents should be able to direct what is considered their child's main quest line.

The parent can set it up so that if their child does not work on the main quest line for a certain amount of time, the child must complete a quest from the main quest line in order to do more side quests

</aside>
