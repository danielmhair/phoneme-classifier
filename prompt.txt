STEP 2 - The task: augmented Chloe-fold pilot (PRD follow-up, approved by
Daniel). The full no-augmentation LOSO baseline is: Chloe (headline) MLP
45.07% / Wav2Vec2-CTC 58.69% / WavLM-CTC 57.90%; no model met the 85% ship
bar; every fold was far above the 10% bug floor. The pilot tests whether
production-style augmentation closes any of the Chloe gap, on the Chloe fold
only, before paying for a full 5-fold augmented rerun. Run in the background
(it's hours long, resumable via caches):

  poetry run python -c "
  from evaluation.harness.loso_runner import run_loso
  run_loso(model_types=['mlp_control','wav2vec2_ctc','wavlm_ctc'],
           ctc_epochs=8, run_id='augmented_chloe_pilot',
           use_augmentation=True, only_folds=['chloe'])"

  (ctc_epochs=8, not 20, is deliberate: the augmented training set is ~6x
  larger, so total gradient updates stay comparable.)
